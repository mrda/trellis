trellis
=======
An experiment in building a CLI for trello

Development Installation
========================

You should run this in a venv. Do something like this:

```
$ python3 -m venv ~/venv3
$ . ~/venv3/bin/activate
$ pip install -U pip
$ pip install .
```

Installing a Release
====================

That's what pypi is for!

```
$ python3 -m venv ~/venv
$ . ~/venv/bin/activate
$ pip install -U pip
$ pip install trellis
```
